== Description ==

The Colby News Theme was custom-built for Colby News in 2021. It uses Gutenberg for content creation and editing, primarily through custom blocks.

Development of this theme and its associated pattern library is done in a private GitLab repository at https://gitlab.com/newcity/colby-news
